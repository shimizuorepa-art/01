# ティアの会連動 お見送り依頼アプリ 提案プロトタイプ

「当日にワンタップで会館へ繋ぐための準備」を目的にした提案用プロトタイプです。

## 内容

- `index.html`: プレゼン用プロトタイプ
- `styles.css`: 画面スタイル
- `app.js`: ワンタップ依頼デモの動作
- `docs/tear-omimokuri-app-design.md`: 設計メモ

## コンセプト

ティア公式の「お急ぎの方へ」で電話時に伺う6項目を、事前登録で埋めておきます。
当日は「お見送りの依頼」ボタンから会館へ正確な初動情報を渡し、ユーザーと会館スタッフ双方の初動負荷を下げます。
